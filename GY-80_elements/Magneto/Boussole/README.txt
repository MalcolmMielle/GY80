Bonjour,

Pour compiler le programme il suffit de faire 'make all' dans un terminal.

Dans ce programme, vous pouvez tourner la caméra en utilisant les touches 2,4,6 et 8. Le zoom, dé-zoom se fait avec + et -. Attention au valeurs limite de représentation, il n'y a pas encore de règles pour les gérer.
Les options de "trajectoire" ou point spécifique" sont demandées dans le terminal au début du programme.
En mode point spécifique, la touche 'e' permet d'enregistrer des points.
Dans les deux modes, la touche 'm' afin d'afficher la trajectoire ou les lignes droites entre les points. ! - Enregistrez quelques points avant de faire un print en mode point spécifique. 

Les options sont accessibles via la touche 'o'. La suite est expliquée dans le terminal.

Pour finir, on quitte le programme via la touche 'q'.

Pour nettoyer le dossier : 'make mrproper'
